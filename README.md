# AI Job Application Agent

An autonomous AI agent that searches for jobs, analyzes job descriptions, matches opportunities against resume skills, and automates portions of the application workflow.

## Features

- Job discovery
- Resume parsing
- AI job matching
- Browser automation
- LinkedIn integration
- Multi-step agent workflows
- LLM-powered decision making
